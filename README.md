# JUST FUCKING USE SVELTE

A humorous website advocating for using Svelte (or frameworks in general) instead of pure HTML/vanilla JS.

Inspired by [justfuckingusereact.com](https://justfuckingusereact.com/) and [justfuckingusehtml.com](https://justfuckingusehtml.com/).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Features

- Dark/light theme toggle
- Responsive design
- SvelteKit for optimal performance
- Same profanity-laden, humorous tone as the original
